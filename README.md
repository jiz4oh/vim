# vimrc

fast setup the vimrc of server

## Install

1. backup your old .vimrc if it is necessary  
    `cp ~/.vimrc ~/.vimrc_bak`
2. just get the file  
    recommend
    
    `curl https://cdn.jsdelivr.net/gh/jiz4oh/vim/vimrc > ~/.vimrc`

    or use git

    ```shell
    git clone https://github.com/jiz4oh/vim.git
    ln -s vim/vimrc ~/.vimrc
    ```

3. Done, enjoy it
